Module contents¶

Tethne is a package for analyzing citation data from the Web of Science. Modules within Tethne can generate a variety of networks, such as bibliographic coupling, citation, author-paper, and co-author networks, using networkx.

analyze The tethne.analyze sub-package provides additional analysis methods not provided by NetworkX, as well as methods for using NetworkX algorithms on an entire GraphCollection.
classes The classes package provides the three most fundamental classes for working with bibliographic data in Tethne: the Paper, the Corpus, and the GraphCollection.
managers
model
networks Methods for building networks from bibliographic data.
persistence
readers Methods for parsing bibliographic datasets.
services Modules for interacting with external web services.
utilities Helper functions.
writers Export networks to structured and unstructured formats, for visualization.
